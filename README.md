## How to Use This Repo

1. Find the article you are reading on Medium
2. Open the matching folder in this repo
3. Clone or download the project
4. Run it in Xcode and follow along

Every project is self-contained. No external dependencies, 
no setup required beyond Xcode.

---

## Requirements

- Xcode 16+
- iOS 17+
- Swift 5.9+
